### Options
 - ** no_model **
   Will not generate Rails model file.
 - ** no_route **
    Will not generate API and default Rails routes
 - ** no_controller **
  Will not generate default Rails Controllers
 - ** no_migration **
 Will not generate default Rails migrations
 - ** no_asset **
 Will not generate CSS and JS files
